***********************
Local Install Workflow:
***********************

1. Open a new terminal

2. Set shot to a show,
>> setshot ccf/home

3. To avoid interference from previoulsy published SPK, make sure to update App's spk version to be something that is not yet published.
    i) open ./build_scripts/spk/rpa_app.spk.yaml
    ii) change the version mentioned in line 1

4. Run the following shell script to build the App spk,
>> ./build_scripts/spk/build_rpa_app_spk.sh

5. Enter into the rpa_app spk,
>> spk env rpa_app/[VERSION]/[HASH]
   When using resolvo: >> spk env --solver-to-run resolvo rpa_app/[VERSION]/[HASH]

[Optional] For spi-rpa_app only:
If gitprod rpa_app plugins need to be loaded, set below env to your local plugins config path
>> setenv RPA_APP_PERIPHERAL_PLUGINS_CONFIGS /net/soft_scratch/users/kjeong/rpa_app-plugins/rpa_app-dev/app_plugins.cfg

6. Install RV Packages and launch spi-rpa_app,
>> ./build_scripts/local_install.sh; ./spi_rpa_app/rpa_app

7. If you want to launch open-rpa_app,
>> ./build_scripts/local_install.sh; ./rpa_app/rpa_app

8. If you want to launch RV with RPA widgets,
>> ./rpa/build_scripts_spi/local_install.sh; ./rpa/open_rv/rv_w_rpa


*********************
SPK Install Workflow:
*********************

1. Open a new terminal

2. Set shot to a show,
>> setshot ccf/home

3. To avoid interference from previoulsy published SPK, make sure to update App's spk version to be something that is not yet published.
    i) open ./build_scripts/spk/rpa_app.spk.yaml
    ii) change the version mentioned in line 1

4. Run the following shell script with the argument "install" to build and install the App spk,
>> ./build_scripts/spk/build_rpa_app_spk.sh install

5. Enter into the rpa_app spk,
>> spk env rpa_app/[VERSION]/[HASH]
   When using resolvo: >> spk env --solver-to-run resolvo rpa_app/[VERSION]/[HASH]

6. Launch spi-rpa_app,
>> rpa_app

7. Launch open-rpa_app,
>> open_rpa_app

Windows install notes:
**********************

>> build_scripts/pkgs/rebuild_pkgs.sh (bash environment assumed while running this - can be done on linux system)

This will create a rpa_app-XX.rvpkg and rpa_app-XX.whl files. Install them using openrv and pip respectively
